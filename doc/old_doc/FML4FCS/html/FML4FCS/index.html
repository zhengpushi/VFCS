What is FML4FCS?

FML4FCS - Formal Method Library for Flight Control System. This is a study for Flight Control System of Unmanned Aerial Vehicles using formal method based on Coq.

Why start this study?

Build a reliable FCS is a big challenge work. We study basic principles and algorithms of FCS formally using Theorem Prover - Coq, and hope to give a full formal verification of this whole system.

Who are we?

We are a team studing the formal verification of mathematical problems in engineering field. We are located in Nanjing, Jiangsu province, China.
